# r0-b008（Beta）
- 新增「屬性」：
  - `nyanko-rpg:max_mana` — 最大魔力（預設 100, 0–1024）
  - `nyanko-rpg:magic_damage` — 魔法攻擊力（預設 0, 0–2048）
  - `nyanko-rpg:magic_speed` — 法術效率（預設 0, 0–1024）
  - `nyanko-rpg:health_regeneration` — 生命恢復速度（預設 0, 0–1024）
  - `nyanko-rpg:mana_regeneration` — 魔力恢復速度（預設 0, 0–1024）
  - `nyanko-rpg:critical_chance` — 爆擊率（預設 0, 0–1.0）
  - `nyanko-rpg:critical_damage` — 爆擊傷害倍率（預設 1.5, 0–100）
  - `nyanko-rpg:drain` — 吸血（預設 0, 0–1.0）
  - `nyanko-rpg:ward` — 結界（預設 0, 0–1024）
  - `nyanko-rpg:ward_toughness` — 結界強度（預設 0, 0–1024）
  - `nyanko-rpg:elemental_fire` — 火元素（預設 0, 0–1024）
  - `nyanko-rpg:elemental_ice` — 冰元素（預設 0, 0–1024）
  - `nyanko-rpg:elemental_electric` — 雷元素（預設 0, 0–1024）
  - `nyanko-rpg:elemental_holy` — 神聖元素（預設 0, 0–1024）
  - `nyanko-rpg:elemental_void` — 虛空元素（預設 0, 0–1024）
